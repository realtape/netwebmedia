## Upload Instructions

### Before you post

1. Use the **native TikTok app** (not TikTok Studio desktop for first publish — the app's algorithm signal on initial post is stronger for organic reach).
2. Upload the same MP4 file from `renders/` — no re-export needed. TikTok accepts 1080x1920 at the same spec Instagram/YouTube gets.
3. After upload, tap **"Auto captions"** before scheduling. TikTok's auto-caption tool boosts completion rate — viewers who read along watch longer. Correct any name/brand errors (e.g., "Net Web Media" → "NetWebMedia," "Apollo" stays, "Claude" stays).
4. Under **"More options"**, set category to **"Business content"**. This enables TikTok's Business Content toggle, which signals the algorithm to serve to a professional/SMB audience rather than Gen Z entertainment.
5. Copy-paste the caption body from the relevant section below. Then paste the 5-6 hashtags on a new line after the body.
6. Do NOT add the caption to the "Add to bio link" field — leave that defaulting to the profile link (netwebmedia.com).
7. Set **Scheduled time** to the CLT time shown per reel. TikTok's scheduler converts to local device time — confirm your phone is set to America/Santiago or manually calculate from the UTC column.
8. Allow comments. Pin the best engagement prompt (from the pinned comment prompts in CAMPAIGN-INSTAGRAM.md) immediately after the post goes live — do not pre-pin, TikTok doesn't support it for scheduled posts.

### Scheduling note
All TikTok drops are **2 hours after the Instagram/YouTube drop** (18:30 CLT IG → 20:30 CLT TikTok). This prevents the platforms' crawlers from flagging near-simultaneous identical posts and gives the IG algorithm its head start.

### CLT to UTC conversion
CLT = UTC-4 (Americas/Santiago standard time, no DST in this window). 20:30 CLT = 00:30 UTC next day.
