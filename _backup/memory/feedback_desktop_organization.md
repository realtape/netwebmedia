**Rule:** Never place files directly on the desktop. Always create a dedicated folder structure and organize files there.

**Why:** Desktop clutter makes it harder to find things and looks disorganized. A structured folder hierarchy keeps things tidy and maintainable.

**How to apply:** When saving files, always ask where they should go or create a logical folder (e.g., `Desktop/Projects/`, `Desktop/Assets/`, `Desktop/Scripts/`) and place files there instead of the root desktop.
