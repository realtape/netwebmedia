## Composition variants

All reels follow the same 30s structure:
- 0–6s: HOOK (full-screen text overlay over b-roll)
- 6–15s: HUD CARD (stat callout — track, position, lap, time)
- 15–23s: BUILD (car/series mention + brand line)
- 23–30s: CTA (`@REALTAPE` handle + "FULL RACE ON THE CHANNEL")
- 0–30s: bottom progress bar (red `#FF1A1A`)

Three archetype color treatments (rotates across the 40):
- **CRIMSON** (default): `#FF1A1A` accent — high energy moments (start, overtakes, finish)
- **AMBER**: `#FFB400` accent — caution moments (mistakes, defensive)
- **CYAN**: `#00D4FF` accent — technical moments (purple sectors, telemetry)

## Rendering plan

Pipeline per reel (~90 sec end-to-end with ffmpeg + hyperframes):
1. **Cut b-roll**: `ffmpeg -ss <start> -i <source> -t 20 -vf "scale=-2:1920,crop=1080:1920,fps=30" -c:v libx264 -crf 18 -an clip-RXX.mp4`
2. **Generate HTML**: template-fill `reel-RXX.html` with hook/HUD/cta vars + clip path
3. **Render**: `npx hyperframes render -q high -f 30 -o reel-RXX.mp4`
4. **Mux audio** (optional): mux original onboard audio or a music bed

Batching: 40 reels × ~90s = ~60 min compute. Run as a script.

**Suggested batch order (for review-as-you-render):**
- Wave 1: R02, R12, R22, R32 (one START per video) — confirms format + branding works on all 4 sources
- Wave 2: R10, R20, R30, R40 (one FINISH per video) — symmetric
- Wave 3: R01–R09, R11–R19, R21–R29, R31–R39 (the rest)
