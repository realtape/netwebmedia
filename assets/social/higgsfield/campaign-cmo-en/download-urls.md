## Re-run a single slide

```bash
# Higgsfield MCP — example: regenerate slide 4 with the same world reference
mcp__16156099-1e4c-4519-9e70-38f64d373711__generate_image
  model:        nano_banana_2
  aspect_ratio: 4:5
  resolution:   2k
  medias:       [{role: "image", value: "ff779eb1-0c54-4f24-8b78-265ea05a304c"}]
  prompt:       "<slide 4's prompt, edited as needed>"
```

The cover (`ff779eb1...`) is the **world anchor** — every slide referencing it preserves lighting, palette, depth of field, and workspace identity. Don't drop the reference unless you want a different world.
