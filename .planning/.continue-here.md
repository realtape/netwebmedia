# BLOCKING CONSTRAINTS — Read Before Anything Else

> Discovered through actual failure this session. Acknowledge before proceeding.

- [ ] **CONSTRAINT: Auto-save daemon fragments history mid-stream.** The Stop hook in `.claude/settings.local.json` runs `_backup/backup.sh` at every turn-end, which does `git add -A` + commit + push. If you're doing surgical git operations on `main`, KEEP THE WHOLE OPERATION INSIDE A SINGLE TURN — do not return control to the user mid-surgery. This session had to consolidate 13 fragmented auto-save commits via `git reset --soft <pre-fragmentation-sha>` + cherry-pick of 2 named commits + force-push-with-lease. Lesson: stage operations atomically across a single turn's tool calls.

- [ ] **CONSTRAINT: InMotion mod_security 406-blocks bare-UA curl.** Every HEAD/GET check against `https://netwebmedia.com/assets/social/*` (or any netwebmedia.com path) MUST use a real Chrome User-Agent (`Mozilla/5.0 (Windows NT 10.0; ...)`) and ideally also `Origin: https://netwebmedia.com` + `Referer:` headers. Without it, you get 406 and conclude the asset is missing. This is documented in CLAUDE.md but bit again this session when first checking PNG availability for IG publish.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Auto-save fragmentation | Stop hook commits + pushes every turn → 13 commits where 4 should exist | blocking | Keep surgical git ops inside a single turn; never return control mid-surgery. Use `git reset --soft <pre-sha>` + cherry-pick + force-push-with-lease for consolidation. |
| Bare-UA asset checks | `curl -I https://netwebmedia.com/...` returns 406 due to mod_security | advisory | Always pass `-A "Mozilla/5.0 ..."` + `-H "Origin: https://netwebmedia.com"` for asset reachability tests |
| Force-push without lease | `git push --force origin main` rewrites remote without checking it hasn't moved | blocking | Always `--force-with-lease=main:<expected-sha>` and check `gh run list` queue first |
| Premature scope expansion | "Create all images and videos" is a 100×-multiplier — burning credits before scope is locked is wasteful | advisory | Always tier the scope (T1/T2/T3) with credit estimates before generating |
